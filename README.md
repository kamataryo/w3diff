# wwwdiff

Node.js only web page diff library.

## Usage

```shell
$ yarn global add wwwdiff
$ wwwdiff https://example.com/a https://examle.com/b > ./sample.png
```

## Sample

```shell
$ npx wwwdiff https://kamataryo.github.io/sample-a.html https://kamataryo.github.io/sample-b.html > ./sample.png
```
